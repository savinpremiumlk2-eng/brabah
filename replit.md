# Prabath-MD Next

A WhatsApp bot based on Baileys.

## Setup on Replit
- The bot uses `start.js` as the entry point.
- Configuration is handled in `config-v2.js` and `config.env`.
- Ensure `SESSION_ID` is valid in `config-v2.js`.
- A workflow "Start Bot" has been configured to run `node start.js`.
